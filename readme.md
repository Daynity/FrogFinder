#### 2. DNA RNA Finder Doc

packages: os, multiprocessing, pandas, glob, sys 



##### 使用

打开terminal, 在frog目录下（里面有自文件夹data存放试验数据）执行命令：

```bash
python DNA_RNA_Finder.py shRNA.csv

```
